# T5 Quantization

Author: Mustafa Cig Gokpinar

PyTorch quantization experiment using FashionMNIST dataset.

## Setup

Python 3.14 recommended.

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Virtual Environment

Create a `.venv` virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

## Output

Generates three visualization files:

- `performance_comparison.png`
- `fmnist_predictions.png`
- `quantization_impact.png`
